Question 1

The main diffrence between web1 , web2 and web3 is that web 1.0 is consider as read-only web targets on content creativity of producer, web 2.0 targets on content creativity of users and producers while web 3.0 targets on linked data sets.

In web 1.0 there is Pushed web, text/graphics based flash, in web 2.0 there is Two ways web pages, wikis, videos, pod casts, shading, Personal publishing, 2D poratals. In web 3.0 there is 3D portals, avtar representation, Interoperable profits, Multi-user virtual environment (MUVEs), Integrated games, education and business, all media flows in and out of virtual web worlds.

another diffrence deals with search engines: In Web 1.0 search engines retrieve macro contents. Search is very fast but many times results are inaccurate. In Web 2.0 search engines retrieve tags with micro contents. Web 2.0 tags everything: pictures, links, events, news, Blogs, audio, video, and so on. In Web 3.0 search engines will hopefully retrieve micro content texts which were tagged automatically. The result could be more precise search.
